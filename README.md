# Kunsh Tap

## How do I install these formulae?

`brew install kunsh/tap/<formula>`

Or `brew tap kunsh/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kunsh/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
